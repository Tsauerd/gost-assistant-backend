# Vibropress AI Assistant

RAG-ассистент для вибропрессованных бетонных изделий (плитка/брусчатка/бордюр/лотки).
Фокус: ответы по ГОСТ/СП с точными ссылками (документ + страница), плюс mix design, оборудование и претензии.

## Модули
1) Нормативка (ГОСТ/СП): требования, допуски, приемка, методы испытаний (с цитатами).
2) Mix design: подбор/оптимизация составов, гранулометрия, влажность, W/C, добавки.
3) Оборудование: гайды/мануалы линий, настройки, обслуживание, troubleshooting.
4) Претензии: чек-листы данных, шаблоны ответов, назначение испытаний.

## Архитектура (коротко)
Raw файлы (PDF/PPTX) → ingestion (парсинг/чанки/метаданные) → индекс (vector + keyword) → hybrid retrieval → LLM ответ.

## Где лежат файлы
- Источники (PDF/PPTX): Google Drive (Shared Drive проекта)
- База: Neon Postgres (+ pgvector)
- Деплой: Render (Web Service + Background Worker)
- Фронтенд: Netlify

## Переменные окружения (Render)
- DATABASE_URL=...
- OPENAI_API_KEY=...
- DRIVE_FOLDER_ID=... (папка с исходниками)

## Запуск локально (пример)
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
